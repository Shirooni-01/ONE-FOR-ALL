# 🚀 ALL FOR ONE

> A collection of useful everyday tools combined into a single web application.

**ALL FOR ONE** brings together multiple productivity utilities in one place. Instead of using different websites or applications, you can access notes, quizzes, password generation, AI chat, QR tools, unit conversion, and text utilities from a single dashboard.

---

## ✨ Features

### 📝 Notes

* Create notes
* Read notes
* Update notes
* Delete single or multiple notes
* Search notes

**Planned (v2)**

* Auto-save notes
* Basic text formatting
* Star important notes
* Password-protected secret notes

---

### 🎯 Quiz

* Take quizzes
* Customize:

  * Number of questions
  * Difficulty
  * Category
* Save quizzes for reattempts
* Category-wise quizzes
* Save quiz scores
* Print or download certificates

---

### 👤 Authentication

* User registration
* Login using:

  * Email + Password
  * Username + Password
* Secure user authentication

**Planned (v2)**

* Profile picture support

---

### 🔐 Password Generator

* Generate passwords (8–64 characters)
* Adjustable password length
* Save last 20 generated passwords
* Add custom names to passwords
* Search saved passwords
* Star favorite passwords
* Delete saved passwords

---

### 🤖 AI Chat

* AI-powered chat using Gemini API
* Copy responses
* Simple chat interface

**Planned (v2)**

* Chat history

---

### 📱 QR Code Tools

* Generate QR codes from text or links
* QR code scanner

---

### 📏 Unit Converter

* Multiple unit conversions
* Formula reference

---

### ✍️ Text Utilities

* Text formatting
* Copy formatted text

---
